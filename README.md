This is a fun playground for different pathfinding algorithms and map generation techniques, including labyrinth generation, where you can generate different maps and visualize how the HPA* algorithm works.

Based on this [discussion thread](https://gamedev.stackexchange.com/questions/192183/how-does-hpahierarchical-pathfinding-a-really-work) about using chunks to restrict the search space for different pathfinding algorithms.

Also inspried by (this)[https://excaliburjs.com/blog/Cellular%20Automata/] apporach for cave generation


The code for map generation and pathfinding was the main focus of this project; the frontend is a bit shit.
